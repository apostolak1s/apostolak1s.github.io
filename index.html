Projects

MICRO-TCP

By implementing our own simpler version of TCP we created a working system that is capable of having clients send message to a server and communicating with other clients. It supports reliable client-server communication through essential TCP features like a three-way handshake, flow control, congestion control, and guaranteed message delivery. Designed for learning and experimentation, MICRO-TCP mirrors the behavior of real-world TCP in a simplified, approachable form.
